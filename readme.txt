== Installation ==

This section describes how to install the plugin and get it working.

1. Install the wordpress plugin first

You can install it via Wordpress Installer or can download the file first then Upload it on 'wp-content' folder

2. Activate the plugin

3. You will see pinterest settings from right side of your wordpress menu.

Configuration Options:

Pinterest Username : Default is "pinterest" - You can change that with your pinterest username

Margin Top - Height & Width : Their you have to Put Numeric Number as per your need.

Image scale : Default is 120 - Change it with a numeric value you prefer.
